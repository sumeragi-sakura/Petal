# Petal Email Client

Petalは、Windows向けの軽量メールクライアントです。古いAL-Mail環境からのデータ移行を支援しつつ、現代的なメール管理機能を提供します。

## 主要機能

- **AL-Mail移行サポート**: AL-Mailの設定、アドレス帳、メールデータを簡単にインポート
- **マルチアカウント対応**: 複数のメールアカウントを管理し、統合受信箱で確認
- **セキュリティ解析**: SPF/DKIM/DMARC認証結果を表示し、なりすましを警告
- **Google連携**: OAuth2認証、セキュリティアラート解析、カレンダー登録機能
- **便利なUI**: トレイアイコン、通知、HTMLプレビュー、自動受信

## インストール方法

1. [GitHub Releases](https://github.com/t-muguruma/almail/releases) から最新の `Petal_Setup.exe` をダウンロードします。
2. ダウンロードしたインストーラーを実行し、画面の指示に従ってインストールしてください。
3. インストール完了後、デスクトップまたはスタートメニューからPetalを起動できます。

## アップデートの確認と方法

Petalは起動時に自動でアップデートを確認します。手動で確認するには：

1. Petalを起動します。
2. メニューから「ヘルプ」→「バージョン情報」を選択します。
3. 現在のバージョンと最新バージョンを比較してください。
4. 新しいバージョンがある場合、[GitHub Releases](https://github.com/t-muguruma/almail/releases) から最新の `Petal_Setup.exe` をダウンロードしてインストールしてください。

## 使い方

### 1. アカウントの追加

- Petalを起動したら、ツールバーまたはメニューから「アカウントの追加」を選択します。
- メールアドレス、受信サーバー（IMAP/POP3）、送信サーバー（SMTP）を入力します。
- GmailなどのGoogleアカウントの場合、「OAuth2 (Google)」タブから「Googleでログインして認証」をクリックして安全に認証してください。
- 必要に応じて、自動受信間隔や通知設定を有効にします。

### 2. AL-Mailデータのインポート

- 古いAL-Mailのデータを移行する場合、メニューから「AL-Mailフォルダをインポート」を選択します。
- インポート先のアカウントを選択し、AL-Mailのメールボックスフォルダ（通常 `C:\ALMail\Mail` など）を指定します。
- 設定ファイル（`ALMAIL.INI`）とアドレス帳（`ALMAIL.ADR`）も自動で読み込まれます。
- 初回起動時にAL-Mailのインストールを検知した場合、インポートを促すダイアログが表示されます。

### 3. メールの受信と表示

- ツールバーの「受信」ボタンをクリックしてメールを受信します。
- 左側のフォルダ一覧からフォルダを選択し、メールリストからメールを選んで内容を表示します。
- HTMLメールは自動でプレビュー表示され、添付ファイルは保存できます。
- セキュリティ情報（SPF/DKIM/DMARC）はメール下部にバナーで表示され、なりすましの疑いがある場合に警告します。

### 4. メールの作成と送信

- ツールバーの「新規作成」ボタンからメールを作成します。
- 宛先、件名、本文を入力し、添付ファイルを追加できます。
- アドレス帳から宛先を選択することも可能です。
- 送信後、送信済みフォルダに保存されます。

### 5. Google連携機能

- Googleアカウントで認証すると、セキュリティアラートメールを自動解析し、デバイス・場所・時間をカード形式で表示します。
- 本文にGoogle Meet URLが含まれる場合、「カレンダーに登録」メニューからGoogleカレンダーにイベントを追加できます。

### 6. その他の設定

- メニューから「設定」でHTML表示のオンオフ、最小化時のトレイ格納、通知の有効化などを変更できます。
- 署名を設定したり、自動受信を有効にしたりできます。

## トラブルシューティング

- **OAuth認証が失敗する場合**: Google Cloud ConsoleでOAuthアプリの設定を確認し、テストユーザーにメールアドレスを追加してください。
- **メール受信ができない場合**: サーバー設定とパスワードを確認し、IMAP/POP3が有効になっているかチェックしてください。
- **AL-Mailインポートがうまくいかない場合**: AL-Mailのフォルダ構造が標準的でない可能性があります。手動でフォルダを指定してください。
